A CLI tool for trading crypto across multiple exchanges directly from the terminal. Binance, Bitfinex, same interface. You configure your API keys, pick a quote currency and timeframe, and execute trades or pull market data without leaving the command line. The whole point was speed. No browser, no UI overhead, just low-latency access to exchange APIs.

## Challenges

Every exchange has its own API. Different authentication schemes, different rate limits, different order formats, different websocket protocols. Making them feel like one unified interface means writing an abstraction layer that hides all of that without losing exchange-specific features.

Latency matters when you're placing orders. The CLI had to minimize overhead between your intent and the API call. That means no unnecessary roundtrips, efficient serialization, and keeping connections alive where possible.

## Tech stack

- Python for the core CLI application
- Click for the command-line interface framework
- Exchange REST and WebSocket APIs (Binance, Bitfinex)
- CSV and JSON for data export
